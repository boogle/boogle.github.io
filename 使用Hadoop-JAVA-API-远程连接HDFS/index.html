0x01 本地操作HDFS

在远程连接HDFS之前，先本地操作一波，便于对HDFS有个更深层的认识。

首先应该对HDFS的概念有个认识，他就是一个文件系统，只不过是适应于hadoop的分布式文件系统，既然是文件系统，其就可以执行跟普通文件系统一样的操作，例如文件操作，目录操作等。
下面演示几个简单的例子。

  • 列出目录文件
    hadoop fs -ls \

  • 创建文件
    hadoop fs -touchz /hello

更多命令参考hadoop HDFS常用文件操作命令
