0x02 后记

但是这个漏洞比较鸡肋的是查询user() database()等，不能进行子查询。原因是因为thinkphp框架使用参数化查询PDO，将参数与查询语句分离，降低了漏洞风险，后门将会针对该框架的PDO进行分析。
